# 『自作エミュレータで学ぶx86アーキテクチャ』

## 概要

このリポジトリは『自作エミュレータで学ぶx86アーキテクチャ』のサンプルコードをManjaro（Linux）環境向けに改変したもののコレクションです．
サンプルコードに同梱されているWindows用のgccやmakeを使わずに，代わりにManjaroのパッケージマネージャでインストール出来るものに置き換えています．

## 学習環境

- OS: Manjaro 21.0 XFCE
- gcc: 9.3.0
